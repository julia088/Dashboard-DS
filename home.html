Sobre o Projeto

Nosso projeto oferece uma plataforma inovadora para monitoramento do consumo de energia. Utilizamos dados em tempo real de medidores para proporcionar insights detalhados, ajudando empresas e residências a gerenciar seus custos e reduzir o impacto ambiental.


Tecnologias Utilizadas


Node.js e Express - Para o backend e rotas de API.

MySQL - Para armazenamento seguro e escalável de dados de consumo.

HTML/CSS e JavaScript - Para uma interface de usuário amigável e responsiva.
